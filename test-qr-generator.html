Centralized Mode:

Simulates CardlessID CDN integration. QR code contains a challenge ID that points to CardlessID service.

  1. Website loads CardlessID JavaScript SDK
  2. SDK creates challenge via CardlessID
  3. User scans QR code containing challenge URL
  4. App fetches challenge details from CardlessID
  5. App sends response to CardlessID
  6. CardlessID callbacks your website
Standalone Mode (Legacy):

Direct callback to website. App sends verification result directly to your callback URL.

Note: This mode is deprecated. Use Centralized Mode for production.
